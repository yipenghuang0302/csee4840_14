This is a robot built with the OpenGL glut libraries. 

To Make: 
	make

To Run:
	./robot_arm

Moving the robot arm (for now): 
	Using keyboard: 
		x: rotates arm on x axis
		X: rotates arm on x axis other way
		y: rotates arm on y axis
		Y: rotates arm on y axis other way 
		z: rotates arm on z axis 
		Z: rotates arm on z axis other way 
		q: quits screen
		Q: quits screen
		o: opens up hand
		c: closes hand
